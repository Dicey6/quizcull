🚀 AUTO PAYMENT SETUP

For one-click payments, set these Vercel Environment Variables (Vercel Dashboard → Project → Settings → Environment Variables):

  • SOLANA_PRIVATE_KEY — base58 private key from Phantom (Settings → Show Private Key)
  • ADMIN_API_SECRET — must match the value in config.js
  • SUPABASE_URL — your Supabase project URL
  • SUPABASE_KEY — your Supabase anon key
  • SOLANA_RPC (optional) — custom RPC endpoint for faster sends

Also run this SQL in Supabase → SQL Editor: ALTER TABLE quizzes ADD COLUMN IF NOT EXISTS reward_sol DECIMAL(18,6);
